# Sungjun Lee Portfolio

A personal portfolio website built with HTML, CSS, and JavaScript, and deployed with GitHub Pages.

## Overview

This portfolio is a space where I present my academic background, interests, writing, and personal projects. Instead of following a conventional portfolio format, I designed the site with a terminal-inspired interface to reflect both my technical interests and personal style.

The website includes pages about me, the courses I have taken at UBC, my book reviews, personal thoughts, bucket list, learning strategy, League of Legends page, and contact information.

## Features

- Terminal-inspired design
- Multi-page portfolio website
- Interactive footer prompt
- Responsive layout
- Deployed with GitHub Pages

## Tech Stack

- HTML
- CSS
- JavaScript
- GitHub Pages

## Design

The portfolio is built around a terminal-style aesthetic with monospace typography, prompt-based section headers, and a consistent dark color palette. I wanted the site to feel simple, structured, and personal.

## Deployment

This portfolio is deployed using GitHub Pages.
